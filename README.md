## Nederlands

### Hoe te spelen?

#### Bedieningselementen
* `W` `A` `S` `D` om voor, links, rechts en achteren te bewegen.
* `Space` om te springen.
* `Left Shift` om te sprinten.
* `M` om *mini-map* in- en uit te schakelen.
* `L` om *full-map* in- en uit te schakelen.
* `Escape` om de muisfocus vrijgeven.
* `Q` om het programma te verlaten.

#### De stand-alone applicatie downloaden (.exe)

Voor gebruikers die het project willen ervaren zonder Unity te installeren:

1. Klik op `Releases` sectie of op de `Droom-Wijk` dat onder `Releases` sectie staat. (Wat u hieronder ziet is gewoon een afbeelding.)

 <img width="387" height="123" alt="Screenshot 2026-01-15 201612" src="https://github.com/user-attachments/assets/1caf3b31-56e1-4924-9392-626a22273492" />
 
2. Als u op `Droom-Wijk` klikt dan opent u altijd de nieuwste versie van de project.
3. Als u op `Releases` klikt dan krijgt u alle versies van dit project.
4. Lees de instructies goed!

#### Het project bouwen vanuit de broncode

Volg deze stappen om het project in Unity te openen en te configureren:

1. **Kloon of download de broncode:** Download de repository met de broncode en pak deze uit.
2. **Toevoegen aan Unity Hub:** Open Unity Hub, klik op "Add" (Toevoegen) en selecteer de projectmap.
3. **Project openen:** Start het project via de Unity Editor.
4. **Pakketten instellen (Package Setup):** Als er een melding verschijnt over ontbrekende pakketten, zorg er dan voor dat de pakketten "Splines" is ge√Ønstalleerd via de Package Manager.

## English

### How to Play

#### Controls
* `W` `A` `S` `D` to move forward, left, right, and backward.
* `Space` to jump.
* `Left Shift` to sprint.
* `M` to toggle the mini-map.
* `L` to toggle the full map.
* `Escape` to release mouse focus.
* `Q` to exit the application.

#### Download the Stand-alone Application (.exe)

For users who want to experience the project without installing Unity:

1. Click on `Releases` or on the `Droom-Wijk` link located under the Releases section.

<img width="387" height="123" alt="Screenshot 2026-01-15 201612" src="https://github.com/user-attachments/assets/1caf3b31-56e1-4924-9392-626a22273492" />
 
2. Clicking on `Droom-Wijk` will always open the latest version of the project.
3. Clicking on `Releases` will show you all available versions of this project.
4. Please read the instructions carefully!

#### Building the Project from Source

Follow these steps to open and configure the project in Unity:

1. **Clone or Download Source:** Download the source code repository and extract the files.
2. **Add to Unity Hub:** Open Unity Hub, click "Add," and select the project folder.
3. **Open Project:** Launch the project through the Unity Editor.
4. **Package Setup:** If prompted about missing packages, ensure the "Splines" package is installed via the Package Manager.
